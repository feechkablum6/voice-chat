# Лобби: Редизайн анимаций аккордеона (Warm Clay)

**Goal:** Улучшить пользовательский опыт в лобби (Warm Clay) путем внедрения современной, плавной и отзывчивой анимации аккордеона. Заменить хрупкий подход на базе `max-height` на `CSS Grid` со "снэповой" пружинистой анимацией.

**Tech Stack:** Нативный CSS (без JS-таймеров для анимации высоты).

## Архитектура анимаций (Accordion)

### Проблема текущего решения:
Текущая реализация (коммит `bd7c0a7`) использует `max-height: 500px`. Это заставляет браузер анимировать всё пространство даже если контента всего на `200px`. Возникают микро-задержки и подергивания содержимого при изменении размеров экрана или контента внутри.

### Решение: CSS Grid `0fr` -> `1fr`
Современный стандарт для аккордеонов. Позволяет анимировать элемент до его "автоматической" высоты контента:
```css
.accordion-body {
  display: grid;
  grid-template-rows: 0fr;
}
.accordion-step.active .accordion-body {
  grid-template-rows: 1fr;
}
```

## Эффекты и Тайминги

1. **Базовый "Bouncy" 타이минг:**
   Будем использовать специальную кривую Безье `cubic-bezier(0.34, 1.56, 0.64, 1)`.
   Она немного "вылетает" за пределы конечной точки (свыше 1) перед тем как стабилизироваться, создавая физическое ощущение пружины (Bounce). Длительность: `0.5s`.

2. **Всплывающий контент:**
   Внутреннее содержимое аккордеона (поля ввода, аватарки) при раскрытии будет:
   - Смещаться снизу вверх: `transform: translateY(-10px) -> translateY(0)`
   - Плавно проявляться: `opacity: 0 -> 1`
   - Задержка контента составит `0.05s`, так что сначала начинает раскрываться блок аккордеона, а затем мгновенно "догоняет" его содержимое, что выглядит очень естественно.

## UI Состояния

Структура UI и цвета шагов (Warm Clay токены) сохраняются из текущей реализации:
- **Открытый шаг (`.active`)**: Получает акцентную рамку (`var(--border-active)`), мягкое оранжевое свечение (glow) и непрозрачность `opacity: 1`. Иконка номера масштабируется.
- **Закрытый/Пройденный шаг (`.completed`)**: Блекнет до `opacity: 0.7`. Покажет мини-превью (выбранный аватар или введенный логин) в правой части (Header) сплюснутое и выровненное по центру. Зеленая галочка заменяет порядковый номер.
- **Заблокированный шаг (`.locked`)**: Недоступен, `opacity: 0.35`.
